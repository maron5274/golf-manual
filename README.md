# 概要
今までのゴルフの研究成果をまとめる

# 1.アイアン
## 1-1.アドレス
### スタンス
わきの開き具合および手首の角度は基本的に脱力したときの自然な形をとるようにする  
プラプラしてみてボールの位置に勝手に戻ってくればOKである  
不自然な姿勢をとるとスイング時に同じ位置に戻ってこない可能性が高い  

立ち位置は気持ち遠目にして、前傾姿勢を取ったときにボールの位置にヘッドが来るようにする  
前傾姿勢をとるときには背骨を曲げずに股関節で前傾する  
前傾姿勢をとることでシャフトが垂直に近い角度を向くので縦振りになり、まっすぐ飛びやすい・ヘッドの重力を利用しやすく飛距離がでる等の効果がある  

脚幅は長いクラブほど広めにとるらしい  
ここは人それぞれであるが基準は決めて毎回同じ脚幅で打てるようにしたい  

### ボールの位置
長いクラブほどボールを前に置くこと  
長いクラブほど重いのでスイング時にヘッドが遅れ、フェイスが開いてミートしてしまうことが多い  
中心より前(腰の回転が十分に入ったタイミング)でミートすることによりこれを防ぐ  

### ヘッドは地面に置かずに"接する"
これはすなわちヘッドが地面からの垂直応力を受けないようにアドレスするということ  
地面からの垂直応力を受けるということは本来の安定位置はより下にあるということで、スイングしたときにダフってしまうことになる  
徐々に腰を倒して前傾していってギリギリ接する瞬間にストップすればよい  
なんなら気持ち浮かせるぐらいでもよいかもしれない(遠心力でクラブが外側に行こうとするので)  

## 1-2.テイクバック
ゴルフは再現性のスポーツである  
再現性をより高めるためには動かす関節をできるだけ少なくして不要な関節を固定することで、変数を減らすことが重要である  
テイクバック時はこのことを心に留めるようにしたい  

### 腰の捻転
腰の捻転は背骨の回転と左ひざの伸縮でつくる  
左ひざを曲げるのは腰の開きをつくるためである  
左ひざを曲げるのはできるだけ真ん前の方向にする  

テイクバックで捻転していく際に前傾の角度を変えないことも重要である  
前傾がゆるんでしまうとアドレス時よりも高い位置でミートしてしまうのでトップの原因になる  
左肩の高さが変わらないことを意識するとよい  
(頭を動かすなと教える人も多いが本質的に重要なのは左肩である)  

### トップの姿勢
アイアンの場合はトップは腰の捻転+右ひじ右手首の伸縮でつくる  
腰を捻転しそのあとで右ひじを曲げればよい  
右ひじ右手首は縦方向に曲げる  
(極座標で考えるならクラブにΦ方向の回転を与えるのが腰の捻転、θ方向の回転を与えるのが右ひじ右手首の伸縮と考えるとわかりやすい)  

あくまで上記はトップの感覚をつかむためのイメージであり、実際のテイクバック時には腰の捻転と右ひじ右手首の伸縮は同時に行われる  
ヘッドを真右に動かす意識でテイクバックしていくとよい  
(人体の構造上真右ではなく少し後ろにテイクバックすることになるが気持ちは真右のつもりでやる)  

## 1-3.スイング
スイング時に意識することは意外と多くない  
捻転時に前に出した左ひざを戻し、あとはクラブの慣性に逆らわないようにスイングすればよい  
アドレス～テイクバックまでで上述の内容を意識できていればこれで大体はうまく打てるはずである  

あとは以下の２点だけ注意しておきたい  
* 腰の回転が遅れるとフェイスが開いた状態でヒットし、ボールが右に飛んでしまうので注意  
* テイクバックのときに維持した前傾角度をインパクトの瞬間まで維持する  

## 1-4.フォロー
フォローについても慣性に従ってクラブを振ればおおよそうまくいく  
腰を最後まで回転しきるということが重要である  
フォローで腰を最後まで回さないとスライスをするような振り方になり、ボールが右に飛びやすくなる  

# 2.ドライバー/ウッド
アイアンと同じ部分は省略してアイアンとの差分のみを述べる  

## 2-1.アドレス
### スタンス
アイアンがわきの開き具合および手首の角度を自然にするのに対し、ドライバー/ウッドは少し無理して遠くに届かせるような意識を持とう  
これはアイアンと異なり横振りであるため、遠心力をより活用するため、というのが理由である  
前傾姿勢も強く取り、膝を軽く曲げ、つま先に力を入れないと前に倒れてしまうぐらいまで行ってよい  

### ボールの位置
長いクラブほどボールを前に置くことはアイアンと同じであり、ドライバー/ウッドはより長いので当然かなり前に置くことになる  
ハンドファーストを計算にいれてボールの位置を決めなければならない  

ウッドなどの長くて重いクラブになるほど、スイングは腰から駆動することも相まって、インパクト時にヘッドよりも手が先行することになる  
これをハンドファーストという  
手とシャフトとヘッドが同一直線になるように構えてしまうと実際のスイング時のハンドファーストの状態と乖離が生じてしまうので、想定外の位置でボールにインパクトしてしまう  
あらかじめハンドファーストになる前提でインパクトの位置を調整するようにしておく  

## 2-2.テイクバック
### 腰の捻転
ここはアイアンと同じ  

### トップの姿勢
ウッド系の場合はアイアンに比べて横振りになるので、右腕の使いかたが複雑になり右ひじ右手首の縦方向の伸縮だけでは実現できなくなる   

アドレスの位置から捻転を行うとクラブが右上に持ち上がるが、トップの位置はその延長線上に置く  
これによりスイング時は腰の捻転で駆動させその慣性を利用してクラブを動かせばアドレスの位置に戻ってこれる  

手首のコックはアイアンほど使わなくてもよい  
アイアンのように縦ぶりではないので必然的に手首のコックも横方向成分が加わることになるが、これがインパクトの際にフェイスが開いてしまう原因になりうる  
クラブが重いためスイング中に手首の角度を素早くもどすのが意外と難しい  
もちろんコックを使うほどヘッドが加速する距離が延びるため飛距離は伸びやすいが、それでシャンクをしてしまうと本末転倒なのでインパクトまでに手首を戻せる範囲に留めよう  

## 2-3.スイング
基本的に意識する点はアイアンと同じである  
ただ、より飛距離を出そうと思ったら手打ちをやめる必要がある  
手打ちにしたくなければ、左ひざを戻し腰の回転が始まった際にその回転に身を任せるのではなく少し無理して腕をトップに近い位置に残す必要がある  
その状態で十分に腰を回せばタメができているはずなので、そこで腕を開放すればヘッドスピードがあがる  

## 2-4.フォロー
フォローもアイアンとほぼ同じであるが、ドライバーのような重いクラブについてはスイング後に手が離れ気味になりやすいので、最後まで右手をしっかり握る意識が大切である  
右手が離れてしまうこともスライスの原因となる   

# 3.アプローチ
100ヤードぐらいのショットをアプローチとよぶこともあるが、ここでは50~60ヤード以内ぐらいのフルスイングをしないショットに限定して説明する  

## 3-1.クラブ
グリーンはフェアウェイやラフよりも芝が短いのでボールが転がりやすい  
ボールが転がると距離の調整が難しくなるので、バックスピンをかけてできるだけボールが転がらないようにしたい  
サンドウェッジのようなフェイスが上を向いているクラブを使用するとよい  

## 3-2.トップに注意！！
アプローチはバックスピンをかけると述べたが、バックスピンをかけるつもりの強さのスイングをしてトップをしてしまったらボールはグリーンを転がって向こう側にオーバーするという悲惨な結果になってしまう  
なので絶対にトップをすることは避けたい  
そんなに短いクラブでミスショットをするのかと思うかもしれないが、簡単な気持ちでやってしまうと意外とトップしてしまう  
これは普通のショットと比べて優しくスイングするため、遠心力や下へ振り下ろす勢いが少なく、クラブの軌道が思ったよりも上を通ってしまうことにより起こる  
以下でトップしないためのポイントも含めて説明するので重点的に意識しよう  

## 3-3.アドレス
### 基本
ウェッジもアイアンには違いないので基本的に上で述べたアドレスの原理から外れることはない  
脚幅は長いクラブほど広めにとると述べたが、アプローチ用のウェッジは短いクラブなのでほぼ完全に閉じてしまってよい  
ボールは両脚の中心より気持ち後ろに置く  
これはトップを防ぐためである  
ボールを前に置くほど最下点を通り過ぎたところでインパクトしやすくなるので、トップの原因になる  
また、アプローチでは飛ぶ向きのずれによる影響が大きいので、いつも以上に丁寧にピンの方向を向けるようにする  

### クラブを握る位置
クラブを短く持った方が当然飛距離が短くなる  
近い距離のアプローチの場合は短く持った方が調整が楽なので短く持つようにする  
スイングのふり幅と併せて、何ヤードの時はどのぐらいのふり幅でどのぐらいの位置を握るのかを決めておくとよい  

## 3-4.テイクバック
テイクバックでどこまで持ち上げるかは距離に応じて異なるが、ハーフスイングの場合もそれより小さい場合も腰の捻転はあまり使わない  
背骨の回転も左ひざの伸縮も多少は行うが、クラブがアドレスの時よりも体の後ろ方向に行かない程度に留める  
イメージとしては回転というよりも左右の重心移動という程度になる  

またアプローチの際はフォローで腰をあまり回し切らないので、インサイド(ボールの右後ろ)から入るとスライスをかけるような打ち方になりやすい  
なのでアウトサイド(ボールの右前)から入ることを意識するとまっすぐに飛びやすい  

## 3-5.スイング
上で述べたようにアプローチではトップをしやすいので、クラブを振るというよりもボールの手前の地面を叩くような意識でスイングするとよい
